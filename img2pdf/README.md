# 图片转PDF工具

一个简单易用的图片转 PDF 工具，支持将文件夹内的多张图片压缩后合成为一个 PDF 文件，提供**命令行**和**图形界面**两种使用方式。

## 功能特性

- 支持常见图片格式：`.jpg` / `.jpeg` / `.png` / `.gif` / `.bmp` / `.tiff`
- 自动按文件创建时间排序，保证页面顺序可预期
- 内置图片压缩（质量 85%），有效减小输出 PDF 体积
- RGBA 图片自动转换为 RGB，避免色彩异常
- 支持多种纸张尺寸：**原尺寸**、**A4**、**B5**、**Letter**、**Legal**
- GUI 版本提供进度条和统计信息（处理张数 / 耗时 / 文件大小）
- 转换完成后自动清理临时文件

## 环境要求

- Python 3.8+

## 安装依赖

```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install img2pdf Pillow
```

## 使用方式

### 🖥️ 命令行版本

```bash
python img_to_pdf.py
```

按提示依次输入：

1. **图片文件夹路径** — 存放待转换图片的目录
2. **纸张尺寸** — 从菜单中选择（1-5）
3. **输出文件名** — 不需要加 `.pdf` 扩展名，默认为 `output.pdf`

### 🪟 图形界面版本

```bash
python img_to_pdf_gui.py
```

操作步骤：

1. 点击 **"浏览"** 选择图片文件夹
2. 点击 **"浏览"** 选择输出 PDF 的保存位置
3. 从下拉框选择纸张尺寸
4. 点击 **"开始转换"**，等待完成

转换成功后会弹窗显示文件名、保存路径、大小、处理张数和耗时。

## 纸张尺寸说明

| 选项   | 尺寸 (mm)          |
| ------ | ------------------ |
| 原尺寸 | 保持图片原始尺寸   |
| A4     | 210 × 297          |
| B5     | 176 × 250          |
| Letter | 215.9 × 279.4      |
| Legal  | 215.9 × 355.6      |

## 注意事项

- 图片默认按文件系统的**创建时间**排序。在某些系统上该时间可能不准确，如有需要可在转换前手动重命名文件来调整顺序。
- 工具会在当前目录创建临时文件夹 `temp_imgs/` 存放压缩后的图片，转换完成后自动删除。
- 压缩质量默认为 85，如需调整可在源码中修改 `compress_image()` 的 `quality` 参数。
